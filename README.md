# StressShield | Intelligent AI Wellness & Stress Management System

**StressShield** is a holistic mental wellness platform designed to help users identify, manage, and mitigate stress. It combines data-driven assessments with Generative AI to provide personalized relaxation strategies, routine planning, and real-time support.

## Core Modules

### 1. Identify Your Stress (Diagnostic Quiz)

- A specialized assessment tool that evaluates user inputs to categorize stress levels (Zero, Low, Moderate, High).
- Implements custom scoring logic to provide instant feedback and directed support.

### 2. AI Chatbot (Wellness Companion)

- Integrated with Google Gemini AI SDK to provide 24/7 empathetic, context-aware support.
- Specialized in helping users optimize sleep schedules, manage anxiety, and improve daily productivity.

### 3. Manage Stress (The Stress-Free Zone)

- **Personalized Routines:** Structured Morning, Daytime, and Evening routine guides.
- **Holistic Resources:** Integrated relaxation music (Nature sounds/Lo-Fi), meditation videos, and downloadable 30-day healthy meal plans.
- **Efficiency Tools:** Quick-access links to Notion life-management templates and routine PDFs.

### 4. Wellness Dashboard

- Visualizes user progress and historical quiz data to monitor mental wellness trends over time.

## Tech Stack

- **Backend:** Node.js (LTS), Express.js
- **Artificial Intelligence:** Google Gemini Generative AI SDK
- **Database:** MongoDB Atlas (Cloud NoSQL)
- **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid), JavaScript (ES6+)
- **Security:** Dotenv, Git/GitHub Environment Management
